# Donkey

Donkey is a planning repository for an AI automatic delivery system aimed at technical infrastructure teams.

The current product direction:

- Target users: technical infrastructure teams that own internal platforms, developer tools, and bugfix/small optimization delivery.
- MVP scope: 2-3 similar repositories, B/D type requirements, automatic delivery to PR with evidence package.
- Product principle: Autonomy-first, risk-gated. Let agents run the workflow, keep humans focused on final delivery acceptance and high-risk confirmations.
