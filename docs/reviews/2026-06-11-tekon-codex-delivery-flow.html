失败修复侧链

风险点 根因判断 落地处理
Codex 认证 internal profile 未生效或凭据未进入 provider。 固定 Codex provider 启动命令为 codex --profile internal。
artifact 输出目录 输出目录未进入 Codex 可写范围。 adapter 在 exec 前受控追加 --add-dir <TEKON_OUTPUT_DIR>。
provider 协议 prompt 对完成边界约束不足。 强化 manifest 完成即停止；禁止节点内 subagent、git push 等动作。
schema drift 真实模型输出 provider-style 字段。 增加归一化和兼容字段，同时保留 schema 边界。
安全扫描 静态测试字符串过像真实密钥。 fixture 改运行时拼接，不削弱生产扫描。
worktree finalize finalize stage 范围过宽。 解析 git status --porcelain=v1 -z，只 stage 真实可提交变更。
RD timeout TDD 技能顺序与 provider artifact protocol 冲突。 RD 节点 manifest 优先，验证交给 Tekon gates。
PR 创建 CommandGateway 把 argv 数据中的 <TEKON_OUTPUT_DIR> 误判为 shell 控制语法。 保持 spawn(..., shell:false)，仅拒绝独立 shell 控制 token。
